        • static Date parse(String date)

          Parse date sent by client (mostly JSON). Expected formats: 2000-03-30, 2000-03-30T22:11:22.123Z , 2000-03-30T22:00:00Z or yyyy-MM-dd'T'HH:mm:ss Assumes all timeZones are UTC //see https://stackoverflow.com/questions/10286204/the-right-json-date-format

          throws:
